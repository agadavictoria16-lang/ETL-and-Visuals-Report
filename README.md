# Sales Data Analytics & Visualization (Excel ETL Project)

## Overview
This project presents an end-to-end sales analytics solution built with Microsoft Excel. It covers data extraction, cleaning, transformation (ETL), dimensional modeling, and interactive dashboard visualization to generate business insights from sales data.

## Dataset
The data is structured using a star schema:
- FactInternetSales – sales transactions and metrics
- DimCustomer – customer details
- DimProduct – product information
- DimTerritory – regional data

## Tools Used
- Microsoft Excel
- Power Query (ETL)
- Charts & Dashboards

## Process
1. Extracted and cleaned raw data  
2. Transformed data using Power Query    
3. Created KPIs and reports  
4. Built interactive dashboards  

## Key Insights
- Sales trends over time
- Regional performance comparison
- Top products and customers

## Skills Demonstrated
ETL | Data Cleaning | Dimensional Modeling | Business Intelligence | Dashboard Design
